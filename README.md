# Navio Clean Starter

Minimaler, frischer Start mit drei Ordnern:
- `frontend/` (Vite + React)
- `api/` (Express API)
- `optimizer/` (Express Service, Stub)

## Schnellstart lokal
```bash
# Terminal 1
cd api
npm install
npm start
# http://localhost:8787/health

# Terminal 2
cd optimizer
npm install
npm start
# http://localhost:8790/health

# Terminal 3
cd frontend
npm install
npm run dev
```
